    Questions Fréquentes

    Remboursement

    Un remboursement est possible uniquement si le code n'a pas été utilisé. Si le code a été utilisé, nous ne pourrons malheureusement pas vous rembourser.

    “Attente de remboursement” signifie que notre système procède à la vérification de la validité du code avant de vous rembourser. Cette étape prend généralement 24h.

    “Remboursé” signifie que nous avons déjà annulé votre transaction et que nous vous avons remboursé. Votre banque peut mettre jusqu'à 7 jours pour re-créditer vos fonds.

    Si vous souhaitez vous faire rembourser un code non utilisé, merci d'utiliser le formulaire ci-dessous, sans oublier de spécifier de quel code il s'agit.

    Produit hors stock, ou non disponible sur le site

    Nous faisons notre maximum pour vous proposer de nouveaux jeux. Si un titre n'est pas disponible, merci d'être patient, nous pouvons l'ajouter très prochainement.

    Il n'est pas possible de vous dire quand un produit sera ajouté ou de nouveau en stock.

    Plusieurs jeux en un achat

    Il n'est pour le moment pas possible d'ajouter plusieurs jeux au panier. Mais cette fonction pourrait voir le jour à l'avenir.

    Quand ma précommande sera disponible ?

    Les précommande peuvent être livrée n'importe quand jusqu'à la fin du jour de la date de sortie (minuit).

    Dans la plupart des cas les codes sont livrés avant la date de sortie pour permettre un préchargement, mais ce n'est pas 100% garanti.

    Problème de performance avec un jeu

    Comme nous ne sommes pas les développeurs du jeu, nous ne pouvons pas prendre la responsabilité d'une éventuelle incompatibilité avec votre système.

    Nous vous conseillons de vérifier que votre ordinateur est assez puissant pour ce jeu, ou de prendre contact directement avec les développeurs du jeu.
